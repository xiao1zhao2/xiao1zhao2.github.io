Hexo+Git 搭建一个个人博客

发表于 2017-07-04 | 分类于 开源工具 | | 阅读次数

使用Github Pages搭建个人网站

  1. Github Pages是GitHub免费提供给开发者的一款托管个人网站的产品，首先需要创建一个GitHub账号
  2. 在GitHub创建一个仓库：username/username.github.io，username是你的账户名，这是特殊的命名约定
  3. 在此仓库提交一个index.html文件，网站内容是在master分支下的
  4. 访问http://username.github.io，就能看到自己的网站了

绑定独立域名

  1. 如果你有自己的域名的话，在之前的仓库点开设置(Settings)，找到GitHub Pages，在Custom domain添加自己的域名，保存(Save)即可
  2. 在你的域名注册提供商那里配置DNS解析，获取GitHub的IP地址，添加A记录即可

使用Hexo搭建博客

  1. 前提
    已安装Node.js和Git

  2. 安装Hexo

    $ npm install -g hexo-cli

  3. 建站

    $ hexo init <folder>
$ cd <folder>
$ npm install

  4. 安装 hexo-server

    $ npm install hexo-server --save

  5. 生成静态文件

    $ hexo generate # 或者 hexo g

  6. 启动本地服务

    $ hexo server # 或者 hexo s
  7. 访问http://localhost:4000/，已经可以看到一篇内置的blog了

部署Hexo到Github Pages

  1. 安装 hexo-deployer-git

    $ npm install hexo-deployer-git --save

  2. 修改配置文件_config.xml
    注：配置文件中，冒号后面一定要加空格

    deploy:
    type: git
    repo: git@github.com:username/username.github.io.git
    branch: master

  3. 执行部署

    $ hexo deploy # 或者 hexo d

  4. 使用git部署
    将我们之前创建的repo克隆到本地，将hexo generate生成public文件夹下的内容copy到本地repo，然后使用git commit命令提交代码到远程repo的master branch上即可

  5. 访问http://username.github.io，能看到Hexo内置的blog已经发布到自己的网站了

  6. 如果使用独立域名访问，需要Hexo也配置域名信息

    $ cd source/
$ touch CNAME
$ vim CNAME # 输入你的域名

Hexo主题相关

  1. Hexo使用Markdown解析文章，支持更换多种主题，本站所用主题是：Next

  2. 切换Hexo主题：
    将自己的主题放在themes文件夹内，并修改_config.yml内的theme设定，即可切换主题。

    theme: next # Hexo默认主题是landscape
